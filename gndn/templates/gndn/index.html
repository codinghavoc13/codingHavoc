Welcome

GNDN is a Star Trek reference. In the original series, there were sets the actors would crawl around that had pipes and conduits everywhere. Many of these pipes and conduits had the letters 'GNDN' written on them. They simply stand for: "Goes Nowhere Does Nothing. That's about what this application is. There is a single table to store basic user information (first and last name, user name, and a hashed password). Use the login page to login and access a third page that is only accessible if you are successfully logged in, otherwise, redirects you back to here.
