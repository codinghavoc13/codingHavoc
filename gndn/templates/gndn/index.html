Welcome

GNDN is a Star Trek reference. In the original series, there were sets the actors would crawl around that had pipes and conduits everywhere. Many of these pipes and conduits had the letters 'GNDN' written on them. They simply stand for: "Goes Nowhere Does Nothing. That's about what this application is. There is a single table to store basic user information (first and last name, user name, and a hashed password). Use the login page to login and access a third page that is only accessible if you are successfully logged in, otherwise, redirects you back to here.

Registering

Registering checks that the username provided isn't already taken. Then, takes a clear text password, generates a salt hash, then hashes the password with that salt before storing in the database.

Logging In

Logging in takes the username provided, checks if that username is in the database and grabs that record, if present. Then the clear password provided when logging in, is passed with the details pulled from the database and sent to validate username/password information. The clear text password is sent with the salt in the user's details and hashed. If that matches the hashed password in the user's profile, you're logged in.
